## Requirements
- `CMake `VERSION 3.2 +

## Dependencies
- `Gtk+3`
- `Gtkmm-3.0`
- `OpenGL`
- `OpenCV`
- `Boost`
- `BLAS`
- `GLEW`
- `GLFW`


## Usage:

#### 1. Create a separate directory "build"
```
$ mkdir build && cd build
```

#### 2. Build 
```
$ cmake ..
$ make
```

#### 3.Run
```
./main_test
```
